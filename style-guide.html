Buttons & Controls

All buttons share border-radius: 0.75rem, a 2px solid black border, and a 0.3–0.4s ease color transition on hover. Background stays in palette on rest; shifts to black or deeper orange on hover.

Motion & Animation

🍈 🍊 🍈 🍊

Falling Papayas

Images fall from off-screen top, spinning a full 360° as they drop. Each instance uses a unique duration (12–45s) and start delay to feel organic.

animation: fall 25s linear infinite;

Typewriter Effect

Text reveals via a CSS steps() width animation with a blinking right border as the cursor. Click to replay — achieved by removing and re-adding the class.

animation: type 1.5s steps(16);

Fade In

Elements enter with a 3s fade that holds at 0 opacity until the 50% mark, then resolves to fully visible — a soft, delayed reveal.

50% { opacity: 0 } 100% { opacity: 1 }

Physics / Verlet

The plant-life canvas uses Verlet integration: gravity 0.01, friction 0.999, with a random breeze impulse every 100–200 frames.

gravity = 0.01 · friction = 0.999

Illustration Style

Flat + Bold Outline

All illustrations use solid fills with a heavy black outline — a sticker or cartoon aesthetic. No gradients, no shadows inside the art.

3-Layer Depth

Each papaya cross-section uses three concentric fills: Skin → Flesh → Gold center. This is the main visual metaphor of the brand.

Energetic & Playful

Shapes are slightly irregular and organic — not perfectly symmetrical. Leans into a hand-drawn, high-energy party vibe.

Seed Dots

Black seed dots inside the fruit are a recurring detail motif. Can be used as texture, pattern, or decorative punctuation in UI.
